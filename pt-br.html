Projetos Práticos

Estação de Inspeção 360° com Inteligência Artificial

Projeto que apresenta uma inovadora estação de inspeção de qualidade automotiva utilizando inteligência artificial e câmeras de alta resolução. A estação é equipada com quatro câmeras posicionadas nas laterais e nas partes frontal e traseira do veículo, que se movimentam de forma sincronizada para obter uma visão abrangente de 360° do veículo.

As imagens coletadas são processadas por um algoritmo de detecção de objetos, que identifica e analisa componentes específicos do veículo. Os dados obtidos são comparados com padrões pré-definidos armazenados em arquivos JSON, contendo informações detalhadas sobre cada modelo de veículo (obtidas a partir do número VIN).

Os resultados finais são apresentados em uma interface gráfica exibida em um monitor de 65" na linha de inspeção de qualidade. Este método de inspeção garante uma verificação precisa e eficiente, reduzindo erros humanos e melhorando a consistência da qualidade do produto final. A integração da IA permite um processo ágil e confiável, tornando a estação uma solução robusta para a indústria automotiva.

    Recomendações de Aprendizado de Python e Programação

    1. Primeiramente, vou te passar um canal gratuito muito completo de conteúdo de programação, tem de tudo que você possa imaginar.

      FreeCodeCamp

    2. O curso mais básico que comecei por volta de 2021 é esse:

      Curso em Vídeo

      Você deve começar com Python mundo 1 primeiro. Lá estão 3 mundos em sequência, como você comentou comigo que já usa a linguagem, este pode ser muito básico, é bem introdutório, mas muito bom para aprender os fundamentos, já que é muito bem editado e didático.

    3. Aqui chegam os cursos da Udemy, que são os mais completos, geralmente custam entre 20-30 reais cada, todo mês tem promoção e ficam todos os cursos da plataforma nesse preço. O curso mais completo de Python na minha opinião é esse:

      Python Completo

      Ou

      Machine Learning para Iniciantes Completo

    4. Tem um site gringo que uso pra aprender qualquer linguagem que é excepcional, porém em inglês também, é necessário aprender inglês pra programar, a maioria dos conteúdos de qualidade é em inglês, a maioria das documentações de bibliotecas/frameworks é em inglês também.

      Exercism

    5. Vídeos de outros canais no Youtube. Tem muitos vídeos disponíveis com todos os conteúdos possíveis. Assim como passei o maior canal sobre programação e computação em geral FreeCodeCamp, vou passar um BR focado só em Python, ele tem lives de Python de tudo que possa imaginar, só ir procurando e explorando as lives salvas no youtube e o repositório do GitHub:

      Dunossauro

    6. ChatGPT, Gemini, etc. Hoje em dia é possível aprender a programar diretamente com essas IAs, elas não somente servem para gerar código, mas para te ensinar o que aquele código faz, ensinar conceitos de programação e conceitos de uma linguagem específica. Use sempre eles.

    7. Documentação de bibliotecas/frameworks. Nas próprias documentações das bibliotecas, já tem explicações completas sobre todas as funcionalidades e muitos exemplos de uso.

    8. GitHub. Não há nada melhor do que ver o código de profissionais.

      Uma das melhores maneiras de aprender é observando como os melhores trabalham e estudando suas práticas. Confira alguns links para ótimos projetos como o kernel do Linux ou TensorFlow, além de outros projetos de código aberto:

      • Kernel do Linux
      • TensorFlow
      • scikit-learn - Biblioteca de aprendizado de máquina em Python
      • PyTorch - Framework de aprendizado profundo
      • Node.js - Tempo de execução do JavaScript
      • Kubernetes - Plataforma de orquestração de contêineres
      • Docker - Plataforma de containerização
      • Apache Spark - Sistema de computação distribuída

    9. Livros. Não utilizei livros para Python puro, somente para inteligência artificial em Python, então não posso recomendar um específico para aprender a linguagem em si, mas estava usando um para aprender Design Patterns em C++ e um outro livro de Rust, e achei bem interessante. A abordagem dos livros é um pouco diferente, ela é mais sequencial, porém acredito que com a quantidade de conteúdo existente hoje em tudo que falei acima, é mais produtivo aprender tudo por lá e caso haja algum livro que te interesse, pegue ele depois pra ler.
