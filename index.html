    Recomendations to Learn Python and Programming

    1. First of all, I'll give you a very comprehensive free channel of programming content, it has everything you can imagine.

      FreeCodeCamp

    2. The most basic course I started around 2021 is this one:

      Curso em Vídeo

      You have to start with Python mundo 1 first. There are 3 worlds in sequence there, as you commented to me that you already use the language, this one might be too basic, it's very introductory, but very good for learning the fundamentals, since it's very well edited and didactic.

    3. Here come the Udemy courses, which are the most complete ones, usually cost between 20-30 reais each, every month there's a promotion and all the courses on the platform are at that price. The most complete Python course in my opinion is this one:

      Complete Python

      Or

      Complete Machine Learning for Beginners

    4. There's a foreign website I use to learn any language which is exceptional, but also in English, it's necessary to learn English to program, most of the quality content is in English, most of the library/framework documentations are in English too.

      Exercism

    5. Videos from other channels on Youtube. There are many videos available with all possible content. Just like I mentioned the biggest channel about programming and computing in general FreeCodeCamp, I'll mention a BR (Brazilian) focused only on Python, it has Python lives about everything you can imagine, just go searching and exploring the saved lives on YouTube and the GitHub repository:

      Dunossauro

    6. ChatGPT, Bard, etc. Nowadays it's possible to learn to program directly with these AIs, they not only serve to generate code, but also to teach you what that code does, teach programming concepts and concepts of a specific language. Always use them.

    7. Library/framework documentation. In the library's own documentations, there are already complete explanations of all the features and many usage examples.

    8. GitHub. There's nothing better than seeing professionals' code.

      One of the best ways to learn is by observing how the bests work and studying their practices. Check out some links to great projects like the Linux kernel or TensorFlow, as well as other open-source projects:

      • Linux Kernel
      • TensorFlow
      • scikit-learn - Machine learning library in Python
      • PyTorch - Deep learning framework
      • Node.js - JavaScript runtime
      • Kubernetes - Container orchestration platform
      • Docker - Containerization platform
      • Apache Spark - Distributed computing system

    9. Books. I didn't use books for pure Python, only for artificial intelligence in Python, so I can't recommend a specific one to learn the language itself, but I was using one to learn Design Patterns in C++ and another book on Rust, and I found it very interesting. The approach of books is a bit different, it's more sequential, but I believe that with the amount of content available today in everything I mentioned above, it's more productive to learn everything there and if there's a book that interests you, pick it up later to read.
